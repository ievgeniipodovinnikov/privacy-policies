# Privacy Policy for Fast API Tester

**Last updated:** April 7, 2026

---

## 1. Introduction

Fast API Tester ("the Extension") is a Chrome extension designed to help developers and QA engineers test API endpoints directly from their browser. This privacy policy explains what data the Extension collects, how it is used, and your rights regarding your data.

---

## 2. Data Collection and Storage

Fast API Tester does **NOT** collect, transmit, or store any personal data on external servers.

All data generated by the Extension is stored **locally on your device** using Chrome's `storage.local` API. This includes:

- Request history (last 3 requests)
- HTTP method, URL, headers, query parameters, and request body
- Response data (status, headers, response body)
- Timestamps of requests

**No data leaves your device.** Everything stays in your browser.

---

## 3. How Your Data Is Used

- **Local storage only:** All data remains on your device. No data is sent to any external server.
- **Copy functions:** When you use the "Copy" buttons (URL, response), the data is copied to your clipboard. The Extension does not log or transmit this data. (cURL copy feature is coming soon in Pro version.)
- **History:** The last 3 requests are stored locally for your convenience. You can clear the history at any time using the "Clear history" button.

---

## 4. Third-Party Services

Fast API Tester does **not** integrate with any third-party services. All API requests are made directly from your browser to the endpoints you specify.

**Important:** When you send a request to an API endpoint, the data is transmitted directly to that server. The Extension itself does not store or process this data beyond displaying it to you.

---

## 5. Data Security

- All data is stored locally using Chrome's secure `storage.local` API
- No data is transmitted to external servers (except the API endpoints you explicitly call)
- Screenshots, logs, and request data are not accessible to anyone except you

---

## 6. Your Rights

Since all data is stored locally on your device, you have full control over it:

- **View:** All requests and responses are visible in the Extension's interface
- **Delete:** You can delete individual history items or clear the entire history using the "Clear history" button
- **Remove:** Uninstalling the Extension will remove all locally stored data

---

## 7. Permissions

The Extension requests the following permissions:

| Permission | Why It's Needed |
|------------|-----------------|
| `storage` | To save request history locally on your device |
| `host_permissions` (`<all_urls>`) | To allow you to send API requests to any URL you specify |
