# Privacy Policy for Fast API Tester

**Last updated:** May 4, 2026

---

## 1. Introduction

Fast API Tester ("the Extension") is a Chrome extension designed to help developers and QA engineers test API endpoints directly from their browser. This privacy policy explains what data the Extension collects, how it is used, and your rights regarding your data.

---

## 2. Data Collection and Storage

Fast API Tester does **NOT** collect, transmit, or store any personal data on external servers.

All data generated by the Extension is stored **locally on your device** using Chrome's `storage.local` API. This includes:

- Request history (last 3 requests for free tier, unlimited for Pro)
- HTTP method, URL, headers, query parameters, and request body
- Response data (status, headers, response body)
- Timestamps of requests

**No data leaves your device.** Everything stays in your browser.

---

## 3. How Your Data Is Used

- **Local storage only:** All data remains on your device. No data is sent to any external server.
- **Copy functions:** When you use the "Copy" buttons (URL, cURL, response), the data is copied to your clipboard. The Extension does not log or transmit this data.
- **History:** Requests are stored locally for your convenience. You can clear the history at any time using the "Clear history" button.
- **Collections (Pro feature):** Saved API requests are stored locally on your device only. They are not transmitted anywhere.

---

## 4. Premium Features (2MinPay)

The Extension offers optional Pro features through **2MinPay** — a payment infrastructure service.

When you choose to upgrade to Pro:

- The Extension calls **2MinPay API** (`https://api.2minpay.com`) to check your payment status
- You are redirected to **Stripe** to complete the payment
- **2MinPay does not collect or store any personal data** beyond your device ID (generated locally) and subscription status
- All payment information is handled by Stripe. Please refer to [Stripe's Privacy Policy](https://stripe.com/privacy) for details

The Pro features include:
- Unlimited request history (instead of 3)
- Collections — save and organize API requests
- Export collections
- cURL copy functionality

You can use the free version without any data being sent to 2MinPay.

---

## 5. Third-Party Services

Fast API Tester integrates with the following third-party services **only if you choose to upgrade to Pro**:

- **2MinPay** (`api.2minpay.com`) — checks subscription status when you open the extension
- **Stripe** — handles payment processing (when you click "Upgrade to Pro")

If you do not upgrade, no data is sent to these services.

All API requests you test are made directly from your browser to the endpoints you specify. The Extension itself does not store or process this data beyond displaying it to you.

---

## 6. Data Security

- All data is stored locally using Chrome's secure `storage.local` API
- No data is transmitted to external servers (except the API endpoints you explicitly call, and optional 2MinPay API for Pro checks)
- Your subscription status is stored locally and verified with 2MinPay servers when you open the extension

---

## 7. Your Rights

Since all data is stored locally on your device, you have full control over it:

- **View:** All requests and responses are visible in the Extension's interface
- **Delete:** You can delete individual history items or clear the entire history using the "Clear history" button
- **Remove:** Uninstalling the Extension will remove all locally stored data
- **Cancel Pro subscription:** Contact our support to cancel your subscription

---

## 8. Permissions

The Extension requests the following permissions:

| Permission | Why It's Needed |
|------------|-----------------|
| `storage` | To save request history and collections locally on your device |
| `host_permissions` (`<all_urls>`) | To allow you to send API requests to any URL you specify |
